# 📊 Contraceptive Method Choice – Data Analysis & Prediction

## Overview
This project explores and analyzes factors influencing contraceptive method choice using a real-world dataset from the UCI Machine Learning Repository. The goal is to understand how demographic and socio-economic variables impact decision-making and to build a predictive model.

---

## Objective
- Analyze patterns in contraceptive use  
- Identify key influencing factors (e.g. age, education, socio-economic status)  
- Build a model to predict contraceptive method choice  
- Support data-driven insights for family planning research  

---

## 📂 Dataset
The dataset is a subset of the 1987 National Indonesia Contraceptive Prevalence Survey.

### Key Features:
- Wife’s age  
- Wife’s education  
- Husband’s education  
- Number of children  
- Religion  
- Employment status  
- Husband’s occupation  
- Standard of living index  
- Media exposure  

---

## 🧪 Project Workflow

### 1. Data Preprocessing
- Cleaned and prepared raw data  
- Handled categorical and numerical variables  
- Checked for inconsistencies and missing values  

### 2. Exploratory Data Analysis (EDA)
- Analyzed distributions and relationships between variables  
- Identified trends influencing contraceptive choices  
- Used visualizations to highlight insights  

### 3. Modeling
- Applied classification techniques (e.g. logistic regression)  
- Evaluated how demographic factors affect predictions  
- Interpreted model outputs to extract meaningful conclusions  

---

## 📈 Key Insights
- Education level plays a significant role in contraceptive choice  
- Socio-economic factors influence access and decision-making  
- Age shows some correlation but isn't as significant
